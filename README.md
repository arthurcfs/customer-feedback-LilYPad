# 📝 Customer Feedback Module

A lightweight customer feedback system using **Laravel API (backend)** and **React (frontend)**.

---

## 📦 Backend (Laravel)

### Setup
```
bash
cd backend
composer install
php artisan key:generate
php artisan migrate
php artisan serve
```

## 📦 Front End (React)

### Setup
```
cd frontend
npm install
npm start
```
